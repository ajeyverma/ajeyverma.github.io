# ajeyverma.github.io

This is a personal portfolio website for Ajay Kumar, showcasing achievements, certifications, and projects.

## Structure

- `index.html`: Main portfolio page
- `Achievements&Certifications.html`: Detailed achievements and certifications
- `assets/`: Static assets
  - `css/`: Stylesheets
  - `js/`: JavaScript files
  - `images/`: Images
- `documents/`: Certificates and documents

## Deployment

This site is hosted on GitHub Pages. Push changes to the `main` branch to deploy.
